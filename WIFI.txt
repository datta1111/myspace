                                                    Wifi adapter issue in Dell:


1)Download zip file 
https://github.com/lwfinger/rtlwifi_new/archive/extended.zip 
2)Unzip the zip file
3)sudo apt update
4)cd Downloads 
5)cd rtlwifi_new #move to rtlwifi adapter directory path
6)make
7)sudo make install
8)sudo /bin/sh -c 'echo "options rtl8723de ant_sel=2" >> /etc/modprobe.d/rtl8723de.conf'  # for better network range only
9)service NetworkManager restart
10) reboot
